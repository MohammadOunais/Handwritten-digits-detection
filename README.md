# Handwritten Digit Recognizer 🧠✍️

This project uses a Convolutional Neural Network (CNN) built with TensorFlow/Keras to recognize handwritten digits from the MNIST dataset.

## 📊 Accuracy Achieved
**Test Accuracy:** 98.98%

## 🧠 Model Architecture
- Conv2D → ReLU
- MaxPooling2D
- Conv2D → ReLU
- MaxPooling2D
- Flatten
- Dense (128) → ReLU
- Dense (10) → Softmax
